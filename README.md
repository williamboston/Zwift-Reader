# Zwift-Reader

Once open, load a fit file recorded in Zwift.

Current Data Ouput:
  - Average Ride Power
  - Average Ride Cadence
  - Average Ride Speed
  - Estimated FTP
  - Ride Time
  - Alt Gained
  - Peak Power
  - Peak Cadence
  - Top Speed
  - Est. Calories
  - Incline Data
  - 1second -> 5hour Critical Power
  - Active/Inactive Breakdown
  - Slope/Gradient Metrics
